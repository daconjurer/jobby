Jobby
======

I once worked on a system that had to interact heavily with a job queueing application.
It was terrible.

This project is inspired by the lessons from those days.

## Quick Start

```bash
# Run tests
make test

# Start MongoDB
make mongo-up

# Run application
make run
```
